## Module Reference

### Core Modules

- `geostats.variogram` - Variogram analysis
- `geostats.kriging` - Kriging interpolation
- `geostats.models` - Variogram models
- `geostats.validation` - Cross-validation

### Advanced Modules

- `geostats.automl` - Automatic model selection
- `geostats.ml` - Machine learning integration
- `geostats.simulation` - Spatial simulation
- `geostats.performance` - Performance optimization

### Utility Modules

- `geostats.io` - Input/output
- `geostats.visualization` - Plotting
- `geostats.config` - Configuration management
- `geostats.workflows` - Pipeline workflows
