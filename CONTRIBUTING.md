# Contributing to GeoStats

**⚠️ IMPORTANT: Run `./dev/check` before pushing!**

See [docs/CONTRIBUTING.md](docs/CONTRIBUTING.md) for full contributing guidelines.

## Quick Start

1. Make your changes
2. **Run `./dev/check` before pushing**
3. Push your changes

The `./dev/check` command runs all pre-push checks (formatting, linting, type checking, and tests) in under 1 minute.

For detailed information, see [docs/CONTRIBUTING.md](docs/CONTRIBUTING.md).
